# Operaciones Matriciales y Vectoriales

Este repositorio contiene un código en **Python** ejecutado en **Jupyter Notebook**, donde se resuelven diversos ejercicios de **operaciones matriciales y vectoriales**. El propósito principal es aplicar los conceptos de álgebra lineal para reforzar la comprensión de matrices, vectores y sistemas lineales, así como practicar la programación científica.

## Contenido del Código
- Suma y multiplicación de matrices.
- Cálculo de inversa y determinante de matrices cuadradas.
- Resolución de sistemas de ecuaciones lineales mediante métodos matriciales.
- Producto cruz y producto escalar de vectores.
- Proyecciones ortogonales de vectores.
- Aplicación del proceso de **Gram–Schmidt** para ortogonalización de vectores.
- Cálculo del espacio nulo de una matriz en forma reducida.

## Objetivo
El objetivo de este proyecto es reforzar los conocimientos de álgebra lineal aplicados en Python, desarrollando habilidades de programación para resolver operaciones matriciales y vectoriales, así como interpretar los resultados de manera analítica y numérica.

## Requisitos
Para ejecutar este código, necesitas:
- Python 3.x
- Jupyter Notebook
- Bibliotecas necesarias: `numpy`, `sympy`, `matplotlib` (opcional para visualización)
